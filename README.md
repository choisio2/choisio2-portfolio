# 🌐 choisio2.com — Personal Portfolio


개인 포트폴리오 웹사이트입니다.

## 🔗 링크

**[choisio2.com](https://choisio2.com)**

## 🛠 Tech Stack

- **Framework**: Next.js 16 + React 19
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **Language**: TypeScript
- **Deployment**: Cloudflare Pages

## 📂 구성

| 섹션 | 내용 |
|------|------|
| Hero | 소개 및 타이핑 애니메이션 |
| About Me | 기본 정보 및 프로필 |
| Archiving | 깃허브, 개발 블로그 |
| Skills | 기술 스택 |
| Projects | 주요 프로젝트 |
| Experience | 대외활동 및 동아리 |
| More About Me | 어학연수, 자격증, 기타 경험 |


## 🚀 로컬 실행

```bash
npm install
npm run dev
```
